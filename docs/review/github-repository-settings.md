# GitHub repository settings checklist

## Repository metadata

Description:

```text
Local desktop OSINT analyst workbench for fusing, qualifying, tracing and explainably scoring observations with provenance, audit trails and reproducible demo data.
```

Recommended topics:

```text
osint
analyst-workbench
local-first
provenance
```

Avoid using one compound topic such as:

```text
osint-map-timeline-audit-trail-electron-dotnet
```

GitHub topics should be separate short keywords for better discoverability.

## Code security

Enable manually in GitHub settings:

- Dependency graph
- Dependabot alerts
- Dependabot security updates
- Private vulnerability reporting, if available
