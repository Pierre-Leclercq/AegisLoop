# AegisLoop v0.2.0-mvp-v1

First public MVP release of AegisLoop.

## Highlights

- Local Electron + React desktop shell
- .NET API and ingestion worker
- RSS/GDELT-style source connectors
- Deterministic offline demo seed
- Dashboard, Map + Timeline, EventCase, Observations and Settings views
- Provenance tracing and append-only audit
- Analyst feedback loop
- Explainable heuristic scoring
- JSON and Markdown EventCase exports
- Offline Natural Earth map layers

## Known limitations

- Demo-oriented MVP
- No production authentication
- No cloud deployment target
- No PDF export
- No advanced GIS engine
- No real-time/SSE demonstration in V1
- Synthetic demo data only
