# Natural Earth 1:50m offline regional basemap

Ces fichiers GeoJSON proviennent de Natural Earth Vector, échelle 1:50m, export GeoJSON public :

- `ne_50m_admin_0_countries.geojson`
- `ne_50m_land.geojson`
- `ne_50m_lakes.geojson`
- `ne_50m_rivers_lake_centerlines.geojson`

Source officielle : <https://www.naturalearthdata.com/>  
Miroir GeoJSON utilisé : <https://github.com/nvkelso/natural-earth-vector/tree/master/geojson>

Natural Earth est publié en domaine public. Ces couches servent le niveau LOD `regional` de Carte + Timeline : elles remplacent le fond monde 1:110m lorsque le `viewBox` SVG descend sous le seuil régional.
